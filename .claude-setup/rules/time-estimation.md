# Time Estimation — AI-Speed (ОБЯЗАТЕЛЬНО)

## Правило #1: Никогда не "дни" если можно "часы"
AI с параллельными агентами работает в 10-20× быстрее человека.

## Формула оценки
```
Человеческое время ÷ 5 (AI ускорение) ÷ кол-во параллельных агентов

Пример: "Человек: 40ч → AI solo: 8ч → AI ×4 agents: 2ч"
```

## Формат ответа
ВСЕГДА показывать:
```
⏱️ Оценка: ~2ч (3 параллельных агента)
  Wave 1 (40мин): [tasks]
  Wave 2 (40мин): [tasks]
  Wave 3 (40мин): [tasks]
```

## Запрещено
- "Это займёт 14 дней" — НЕЛЬЗЯ. Разбей на waves с параллельными агентами.
- "Несколько дней" — НЕЛЬЗЯ. Конкретные часы.
- "Зависит от..." — НЕЛЬЗЯ. Дай best case + worst case в часах.

## Когда реально нужно время
- Ожидание внешних API (DNS propagation, SSL) — указывай отдельно как "ожидание"
- Ручные действия Дмитрия (регистрация, оплата) — указывай отдельно
- Деплой + тестирование — включай в оценку
