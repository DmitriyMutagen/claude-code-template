# Auto-Coaching — AI дотягивает до Elite уровня (ОБЯЗАТЕЛЬНО)

## Когда срабатывает
- В /good-morning и /daily-review — секция коучинга
- После каждой крупной задачи — мини-анализ
- При запуске `python3 engine.py` — competitive ranking + coaching

## Что анализировать
1. **Метрики ниже Senior** (percentile < 50) → конкретные действия для роста
2. **Регрессии** — метрика упала vs прошлая неделя → предупреждение
3. **Недоиспользованные инструменты** — MCP/skills/hooks которые есть но не юзаются
4. **Паттерны ошибок** — из reflexion.md и CL v2.1 instincts

## Формат коучинга
```
🎓 Coaching:
  ⬆️ Рост: тесты выросли с 0 до 87/день за неделю (+∞%)
  ⚠️ Внимание: 0 деплоев за неделю (Junior уровень)
    → Действие: задеплой Aragant или Bionovacia сегодня
  💡 Совет: используй /e2e после каждой фичи — это +40 XP и рост Quality ранга
  📊 До Elite: нужно +26% по деплоям и +15% по тестам
```

## Правила коучинга
- Говорить КОНКРЕТНО что делать, не абстрактно
- Привязывать к XP и рангу — мотивация через геймификацию
- Максимум 3 совета за раз (не перегружать)
- Хвалить за рост: "заебись, тесты с 0 до 87 — God Tier!"
- Если метрика Junior → приоритет #1 для роста
