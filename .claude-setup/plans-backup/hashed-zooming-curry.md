## Фаза 3: Система скоринга

### 3.1 Мультимерный скоринг на основе обучения
**Файл:** `src/tools/nep_scoring.py` (НОВЫЙ)

**Измерения скоринга (из всех эфиров):**

| Эфир | Тема | Метрики скоринга |
|------|------|-----------------|
| 1 | Результат | Заполненность карточки, качество контента |
| 2 | Внешний трафик | Источники трафика, эффективность каналов |
| 3 | Формирование трафика | Охват запросов, глубина семантики |
| 4-5 | (уточнить) | |
| 6 | Карта запросов | Покрытие частотных/средних/низких, кластеризация |
| 7 | Тренды | Попадание в тренды, сезонность |
| 8 | Дубли | 5+ различий на каждом слое (уже реализовано) |
| 9 | SEO | Наименование, описание, характеристики, ключевые слова |
| 10 | Продвижение | 3 стратегии, экономика, рекомендации |
| 11 | Конверсионные цепочки | Покрытие намерений/мотивов, путь покупателя |

**Модель скоринга:**
```
Общий балл =
  Карточка (30%):
    - Наименование (10%): длина, ключевые слова, первые 60 символов
    - Описание (8%): длина, SEO, USP
    - Характеристики (7%): заполненность, точность
    - Фото (5%): количество, разнообразие, инфографика

  Антидубль (25%):
    - Различия хар-к (10%)
    - Различия фото (8%)
    - Различия текста (7%)

  SEO (20%):
    - Покрытие запросов в title (8%)
    - Покрытие в description (6%)
    - Покрытие в характеристиках (6%)

  Конверсия (15%):
    - Намерение × Мотив покрытие (8%)
    - CTR прогноз (4%)
    - Конверсионная цепочка (3%)

  Экономика (10%):
    - Маржинальность (4%)
    - Рекламный ROI (3%)
    - Цена vs рынок (3%)
```

### 3.2 Чек-листы

- Приоритеты действий (что исправить в первую очередь)
