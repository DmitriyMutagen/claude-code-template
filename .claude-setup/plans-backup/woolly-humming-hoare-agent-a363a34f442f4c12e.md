5. **Детальные составы** -- для Bionordiq полные рецептуры не публикуются в открытых источниках. Для seed-данных потребуется внутренняя документация Бионовации (рецептурные карты, технические условия).

6. **Цены на маркетплейсах** -- динамические, зависят от промоакций. Для актуальных данных нужен парсинг через WB API / Ozon API (настроены MCP-серверы).
